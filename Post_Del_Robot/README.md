# Post Del Robot

**删帖机器人**

## 安装

* [【点击安装】](https://github.com/FirefoxBar/userscript/raw/master/Post_Del_Robot/PostDelRobot.user.js)

## 功能

* 按用户ID或关键词批量删帖

* 注意：一次性删除有最大数量限制，可分多次使用
