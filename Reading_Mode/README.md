# Reading Mode

给一些网站添加阅读模式的功能

支持网站：

* CSDN博客

* 新浪博客

* 网易博客

* 搜狐博客

* 天涯博客

* 科学网博客

* ChinaUnix

* cnblogs

[【点击安装】](https://github.com/FirefoxBar/userscript/raw/master/Reading_Mode/Reading_Mode.user.js)
